## Getting Started

If you want to run this project locally

### Prerequisites

Use the latest version on npm
```sh
npm install npm@latest -g
```

### Installation

1. Clone the repo
  ```sh
  git clone https://github.com/luisalonso97/blackjack-count
  ```
2. Install NPM packages
  ```sh
  npm install
  ```
3. Run the development command
  ```sh
  npm run dev
  ```
